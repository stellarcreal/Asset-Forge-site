    Preview Renders¶

    Asset Forge renders custom asset preview thumbnails with full control over camera and lighting. Inspect the setup before committing, then batch render in Quick or Eevee mode.

    Environment Setup¶

    Lighting Type¶

    • Sun — A directional sun light with configurable energy and rotation.
    • HDRI — An environment texture for image-based lighting. Select an .hdr or .exr file from disk.

    Background¶

    Configure the world background for your preview renders. Options depend on the lighting type selected.

    Transparent Background¶

    Enable transparent background for setup previews when you want the object rendered without a visible world behind it.

    Camera Setup¶

    Projection¶

    Choose the camera projection type for preview renders.

    Padding¶

    Controls how much space surrounds the object in the preview. Higher padding means the object appears smaller with more breathing room.

    Camera Angle Presets¶

    Quick presets for common camera angles. Select a preset to snap the camera to a predefined position.

    Setup Preview¶

    Setup preview lets you inspect the camera and lighting rig before committing to a full render.

    1. Configure your environment and camera settings.
    2. Click Refresh Setup Preview to generate a sample render.
    3. Review the result and adjust settings as needed.
    4. Refresh again after any changes.

    Note

    Setup preview is a static sample render — it does not update live. Refresh it every time you change lighting or camera settings.

    Render Modes¶

    Quick¶

    A fast Workbench-based render path. Produces approximate previews in minimal time. Best for bulk operations where speed matters more than visual fidelity.

    Eevee¶

    A full Eevee render using the add-on's render rig. Produces higher quality previews with proper lighting, shadows, and materials.

    • Supports cancel with Esc during rendering.
    • Takes longer than Quick, especially for large batches.

    Batch Scope¶

    Selected¶

    Render previews for the currently selected assets. Choose whether Selected reads from:

    • Outliner / Viewport — Uses the selection in the 3D viewport or outliner.
    • Asset Browser — Uses the selection in the Asset Browser.

    Collection / Catalogue¶

    Render previews for all assets in the active collection or asset catalogue.

    All¶

    Render previews for every asset in the file.

    Clear Asset Previews¶

    Removes custom preview images from selected assets, reverting them to Blender's default auto-generated previews.

    Tip

    Use Dry Run before batch rendering to preview which assets will be processed.
