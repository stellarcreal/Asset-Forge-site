    Asset Forge¶

    Build material assets from texture folders, mark object assets, render custom previews, and dry run everything first, all from the Asset Browser sidebar.

    Asset Forge is a Blender 5.1 add-on that simplifies and automates common asset-related tasks directly inside the Asset Browser.

    Superhive page not up yet Documentation

    What It Does¶

    • Material Assets

      Build PBR material assets from texture folders with a single click. Scans for common map types with customizable naming keywords, sets up shader graphs, and registers everything as assets. Supports UDIMs, overwrite policies, and manual map overrides.

    • Object Assets

      Mark Selected Objects, Selected Collection, or All in one action, with automatic catalog writing and hierarchy placement controls for parents, children, and assemblies.

    • Preview Renders

      Render custom asset thumbnails with configurable camera, lighting, thumbnail size, and selection scope. Inspect the setup before committing, then batch render in Quick or Eevee mode.

    • Dry Run & Reports

      Preview what every action will do before applying. Results appear in a report panel with expandable details and issue summaries.

    Requirements¶

    Requirement Details
    Blender 5.1 or newer
    Saved file A saved .blend file is needed for object asset catalogs and rendered preview output

    Support for earlier Blender versions is planned for a future update.

    Installation¶

    1. Download the release .zip.
    2. In Blender, open Edit > Preferences.
    3. In Get Extensions or Add-ons, open the menu and click Install from Disk...
    4. Select the downloaded .zip.
    5. Enable Asset Forge.

    The add-on appears in the Asset Browser sidebar under the Asset Forge tab.
